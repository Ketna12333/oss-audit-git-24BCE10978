## Scripts

Script 1 - System Identity Report
Shows Linux system details like kernel, user, uptime, and distro.

Script 2 - FOSS Package Inspector
Checks if Python is installed and shows its information.

Script 3 - Disk and Permission Auditor
Displays size and permissions of important system directories.

Script 4 - Log File Analyzer
Counts how many times a keyword appears in a log file.

Script 5 - Open Source Manifesto Generator
Creates a personal open-source statement and saves it to a file.

## How to Run

Make scripts executable:
chmod +x *.sh

Run scripts:
./01_system_identity.sh
./02_package_inspector.sh
./03_disk_auditor.sh
./04_log_analyzer.sh
./05_manifesto_generator.sh
